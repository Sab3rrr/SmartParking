# 智慧社区停车服务业务管理
## 项目结构说明
SmartCommunity/    
├── docs/                      # 项目文档   
│   ├── flowcharts/               # 业务流程图  
│   ├── database_design/          # 数据模型设计说明（居民/外来人员/停车等）    
│   └── 项目结构说明.md             # 项目结构说明    
├── config/                    # 配置文件    
│   └── config.py                 # 应用全局配置  
├── src/                       # 核心源代码     
│   ├── models/                # 数据模型层（ORM模型+数据验证）  
│   │   ├── resident_model.py     # 居民信息ORM模型  
│   │   └── resident_pydantic.py             
│   ├── ui/                    # 界面层  
│   │   ├── ui_admin/             # 管理员界面    
│   │   ├── ui_user/              # 居民用户界面   
│   │   └── ui_login.py           # 登录界面    
│   ├── db/                    # 数据库操作层  
│   │   ├── parking_system.db     # 停车系统数据库  
│   │   └── parking.db            # 停车数据库    
│   ├── utils/                 # 工具函数层  
│   │   └──utils.py              # 时间处理（如停车计时）     
│   └── main.py                # 应用入口    
├── tests/                     # 测试目录    
│   ├── test_resident.py  
│   ├── test_parking.py  
│   └── conftest.py            # 测试配置    
├── requirements.txt           # 依赖清单      
└── README.md                  # 项目说明
